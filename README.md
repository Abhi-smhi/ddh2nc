# ddh2nc

Combine a large no.~ of DDH output files to single NetCDF dataset. Multiple
domains allowed. Currently only for vertical profiles. Runs in parallel.


## Prerequisites
  - Python 3.10+
  - ddh toolbox [https://github.com/UMR-CNRM/ddhtoolbox]
    - ensure tools ´lfac´ and ´ddhr´ are on ´$PATH´

## Installation
  - In your preferred python  environment: ´pip install -r requirements.txt´

## Usage
  - On Atos:

  - ´./ddh2nc.py´
